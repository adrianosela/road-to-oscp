## LinPEAS

A privilege escalation vector detection script for Linux ([docs](https://github.com/carlospolop/PEASS-ng/tree/master/linPEAS)) ([releases](https://github.com/carlospolop/PEASS-ng/releases)).

Usage: (just invoke the binary for the script to run)

```
linpeas_linux_amd64
```

can download with cURL and pipe directly to a shell

```
curl -L https://github.com/carlospolop/PEASS-ng/releases/latest/download/linpeas.sh | sh
```
